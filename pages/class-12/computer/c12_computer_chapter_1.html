Unit 1: Database Management System (DBMS)

Last Updated: Dec 2025 • Unit 1

Understanding the core concepts of data, information, databases, and database management systems.

1.1 Introduction

A Database is a collection of related data. By data, we mean known facts that can be recorded and that have implicit meaning. For example, names, telephone numbers, and addresses of people you know.

A Database Management System (DBMS) is a software package designed to define, manipulate, retrieve and manage data in a database. Examples include MySQL, PostgreSQL, Oracle, and Microsoft SQL Server.

Exam Tip: Remember the difference between Data (raw facts) and Information (processed data).

1.2 Advantages of DBMS

  • Data Independence: Application programs are independent of the details of data representation and storage.
  • Efficient Data Access: A DBMS utilizes a variety of sophisticated techniques to store and retrieve data efficiently.
  • Data Integrity and Security: Data is accessed through the DBMS, which can check authorization and integrity constraints.
  • Concurrent Access: Multiple users can access the database at the same time without conflict.

1.3 Data Models

Data models define how the logical structure of a database is modeled.

Relational Model

Data is organized in tables (relations) with rows and columns. This is the most popular model.

Hierarchical Model

Data is organized into a tree-like structure. Each record has a single parent.
